Project 2: CineTracker - Personal Movie Catalog

A dedicated movie database interface designed to catalog and track favorite cinema listings.

Project Overview

In alignment with my core hobby of watching movies, I constructed CineTracker. This project serves as an interactive movie logger catalog where users can review reviews, track watchlists, and sort titles by genre, director, and release year.

This project focuses heavily on visual grids, card alignments, and dynamic styling patterns. Integrating media layouts effectively was key to ensuring poster graphics loaded beautifully without causing layout shift.

Technical Highlights

  • Responsive Flexbox Grid: Rendered film posters dynamically to fit mobile devices perfectly.
  • Audio & Video Tags Integration: Native semantic HTML5 media tags for trailer audio playback configurations.
  • Custom Interactive Review Form: Standard-compliant validation parameters allowing structured film grading.

Project Details

  • Role: Front-End Developer
  • Tech Stack: HTML5, CSS Grid, Bootstrap 5
  • Target Audience: Cinephiles
  • Status: Completed
